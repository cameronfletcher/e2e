A trivial end-to-end web solution.

Get Started
Written in C# and using Nancy running on ASP.NET.
Uses Skeleton; a dead simple, responsive boilerplate.
Uses Microsoft SQL Server for persistence.
What could this be used for?

Not a lot.

This is a demonstration of different web and persistence technologies working together. Specifically, this solution incorporates a static web-based front-end UI which communicates with an HTTP web service running in a stand-alone OWIN host that may be installed as a Windows service.

This code is hosted on GitHub if you're interested in the implementation details.
